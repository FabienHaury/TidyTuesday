## Introduction and problematic   
The world of data is an ever-growing field. Covering the full spectrum from gathering and cleaning data to analysis, one side is often left aside: data visualisation.  
All the work done to make the data speak can be lost if the way to communicate it makes it incomprehensible.  
That is why this week, the focus is on growing skills in data visualisation and on how to make graphics that not only look good but also help answer questions more clearly.
